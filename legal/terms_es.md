# Términos y Condiciones

_Última actualización: 10 de febrero de 2026_

Bienvenido a **Knowting**. Al acceder o utilizar nuestra plataforma, aceptas quedar vinculado por estos Términos y Condiciones ("Términos"). Si no estás de acuerdo, por favor no utilices la Plataforma.

---

## 1. Definiciones

- **Plataforma**: La aplicación web proporcionada por Knowting.
- **Usuario**: Cualquier persona que accede o utiliza la Plataforma.
- **Contenido**: Todo material subido, creado o compartido en la Plataforma, incluyendo tests, preguntas, materiales de estudio (vídeos, PDFs, imágenes), y programas.
- **Creador**: Un Usuario que crea o publica tests o programas.
- **Estudiante**: Un Usuario que realiza tests o participa en programas.
- **Programa**: Una colección de tests agrupados para su estudio conjunto.

---

## 2. Elegibilidad

Debes tener al menos **16 años** para utilizar la Plataforma.
Al usar la Plataforma, declaras que tienes capacidad legal para aceptar estos Términos.

- Eres responsable de mantener la seguridad de tu cuenta de Google.

Nos reservamos el derecho de suspender o terminar cuentas que violen estos Términos.

## 4. Tipos de Usuario y Funcionalidades

La Plataforma ofrece diferentes niveles de acceso:

- **Usuario Gratuito**: Puede crear tests, añadir preguntas y materiales, e invitar colaboradores.
- **Usuario Premium**: Además de lo anterior, puede crear programas (colecciones de tests).
- **Administrador**: Acceso completo incluyendo gestión de usuarios.

### 4.1 Colaboración

Los propietarios de tests y programas pueden invitar a otros usuarios con diferentes roles:

- **Estudiante**: Solo puede realizar el test.
- **Invitado**: Puede ver el contenido y realizar el test.
- **Revisor**: Puede editar preguntas.
- **Administrador**: Puede gestionar colaboradores.

- La realización de tests **no** garantiza certificación, empleo o cualificación profesional.

## 8. Visibilidad del Contenido

Los tests y programas pueden tener diferentes niveles de visibilidad:

- **Público**: Visible y accesible para todos los usuarios.
- **Restringido**: Todos pueden hacer el test, pero los materiales solo son visibles para colaboradores.
- **Privado**: Visible para todos, pero solo accesible con invitación.
- **Oculto**: Solo visible y accesible para usuarios con acceso explícito.
